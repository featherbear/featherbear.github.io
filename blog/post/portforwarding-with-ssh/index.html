Portforwarding with SSH

Posted Nov 12, 2018

$> ssh [SERVER] -L srcAddr:srcPort:dstAddr:dstPort listens on srcAddr:srcPort which results in [SERVER] connecting to dstAddr:dstPort
> Useful for connecting to a device in a network

$> ssh [SERVER] -L srcPort:dstAddr:dstPort
[SERVER] listens on srcPort which is connecting to connecting to dstAddr:dstPort
